This Python text adventure transforms traditional Rock Paper Scissors into a sarcastic meme filled command prompt experience. Players are immediately greeted by a highly self aware narrator who casually guides them through a fake matchmaking process. You will face off against a quirky roster of simulated internet opponents with absurd usernames like ipadkid2026 and joshproz69.
​The core gameplay operates on a first to three wins match format, complete with dramatic and humorous commentary for every single move you execute. Beyond standard play, users can select an AFK option to trigger hilarious, completely randomized outcomes or simply resign in total defeat.
​Additionally, the script features a fully functional progression system utilizing JSON files. This automatically saves and loads your custom username, total win count, and loss record across multiple gaming sessions. Hidden menu easter eggs constantly reward curious players, making this lightweight console program a delightfully funny twist on a classic.
